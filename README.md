# 项目工程化


## lerna
为`packages`下面每个包安装本地或远程包
```bash
npx lerna add <package>[@version] --dev or --prod
```
命令:  
`lerna publish`   
`lerna version`   
`lerna bootstrap`  
`lerna list`  
`lerna changed`  
`lerna diff`  
`lerna exec`  
`lerna run`  
`lerna init`  
`lerna add`  
`lerna clean`  
`lerna import`  
`lerna link`  
`lerna create`  
`lerna info`
